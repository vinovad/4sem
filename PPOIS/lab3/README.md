## Описание игры

**Jewel Quest** — головоломка типа «три в ряд». Игрок обменивает местами соседние камни, чтобы создавать цепочки из 3 и более одинаковых элементов.

### Режимы игры

| Режим | Описание |
|-------|----------|
| **На время** | Набирайте максимум очков за 2 минуты |
| **На очки** | Пройдите 3 уровня, достигая цели по очкам |

## Реализованные требования

### Обязательные (из задания):

#### 1. Два режима игры (п. 1)
- **На время** (`time_mode`): игрок набирает очки за 120 секунд
- **На очки** (`score_mode`): 3 уровня со своими целевыми значениями (1000 / 3000 / 6000 очков)

#### 2. Три уровня для режима «На очки» (п. 2)
| Уровень | Название | Цель | Доступные цвета |
|---------|----------|------|-----------------|
| 1 | Рубиновый сад | 1 000 | 5 цветов |
| 2 | Изумрудные пещеры | 3 000 | 6 цветов |
| 3 | Сапфировые высоты | 6 000 | 7 цветов |

Параметры уровней хранятся в `configs/levels.json`.

#### 3. Анимации (п. 3)
- **Появление камней**: масштабирование от 0 до 1 при спауне
- **Обмен камней**: плавное перемещение с `ease_in_out`
- **Уничтожение**: прозрачность + увеличение масштаба
- **Падение**: гравитация при заполнении доски
- **Пульсация**: выбранный камень мягко пульсирует
- **Частицы**: система частиц при совпадениях
- **Всплывающие очки**: ScorePopup поднимается вверх и исчезает
- **Анимация кнопок**: плавный scale-эффект при наведении

#### 4. Три вида специальных камней с эффектами (п. 6) 
| Камень | Эффект |
|--------|--------|
| 💣 **Бомба** | Уничтожает все камни в радиусе 1 клетки |
| ⚡ **Молния** | Уничтожает всю строку и весь столбец |
| ⭐ **Звезда** | Уничтожает все камни того же цвета на доске |

### Общие требования:

| Требование | Реализация |
|-----------|------------|
| Событийно-ориентированное программирование | `pygame.event` в `game.py`, обработчики `handle_event` |
| Конфиги во внешних файлах (JSON) | `configs/settings.json`, `levels.json`, `gems.json` |
| Меню: начать / рекорды / справка / выход | `_draw_menu()`, `_draw_highscores()`, `_draw_help()` |
| Звуки и фоновая музыка | `SoundManager` — поддерживает загрузку внешних MP3/WAV файлов или генерацию через numpy |
| Диалог нового рекорда | `Dialog` с полем ввода имени |
| Анимации | Система частиц, плавное перемещение, прозрачность |

## Игровая механика

### Матчинг
Игра ищет совпадения горизонтально и вертикально:
- 3 в ряд = **100 очков**
- 4 в ряд = **250 очков**
- 5+ в ряд = **500+ очков**

### Комбо
Каждое последующее совпадение без хода игрока увеличивает счётчик комбо.

### Гравитация
После уничтожения камни падают вниз, сверху появляются новые.

### Перемешивание
Если на доске нет возможных ходов, она автоматически перемешивается.

## Конфигурационные файлы

### `configs/settings.json`
```json
{
  "window": { "width": 800, "height": 700, "fps": 60 },
  "board": { "rows": 8, "cols": 8, "cell_size": 70 },
  "scoring": { "match_3": 100, "match_4": 250, "match_5": 500 },
  "animation": { "swap_speed": 8, "fall_speed": 12 }
}
```

### `configs/gems.json`
Описывает цвета и названия обычных камней, а также конфигурацию трёх видов специальных камней с их эффектами.

### `configs/levels.json`
Описывает 3 уровня: название, цвет фона, список доступных типов камней, целевой счёт, вероятность спецкамней.
